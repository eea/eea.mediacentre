A media file can be one of several types, e.g. interview and map. This can be
specified on any file object using an adapter.

  >>> from zope.app.folder.folder import Folder
  >>> from zope.interface import alsoProvides
  >>> from eea.mediacentre.interfaces import IMediaType
  >>> from p4a.video.interfaces import IMediaActivator, IVideoEnhanced

  >>> folder = Folder()
  >>> folder.reindexObject = lambda: None

Objects have no media type by default

  >>> IMediaType(folder).types
  Traceback (most recent call last):
  ...
  TypeError: ('Could not adapt', ...

When an object is activated it automatically get's the 'other' type.

  >>> activator = IMediaActivator(folder)
  >>> activator.media_activated = True
  >>> IMediaType(folder).types
  ['other']

Let's make sure adding types works.

  >>> IMediaType(folder).types = ['interview', 'video']
  >>> IMediaType(folder).types
  ['interview', 'video']

If the object is media inactivated it shouldn't have any type anymore.

  >>> activator.media_activated = False
  >>> IMediaType(folder).types
